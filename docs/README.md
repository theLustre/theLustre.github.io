# theLustre Docs

这里用来同步 Obsidian/笔记系统的原始 Markdown。后续可以把这些文件接入 Astro Content Collections 或 Pagefind，实现全文搜索。

## 结构规划

- `docs/notes/`：零散想法、周报。
- `docs/projects/`：项目复盘、上线记录。
- `docs/handbook/`：流程、脚本、CLI 命令备忘。

当前仓库默认不会直接渲染这些 Markdown，你可以根据需要把它们导入 Astro 组件或单独部署。
